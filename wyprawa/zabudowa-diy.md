## [EN] Project Description
Camper build for Hyundai Starex 4x4. The main goal was to create a comfortable sleeping area while maintaining 4 seats for driving.

* **Configuration:** 2 front seats + 2 swiveling middle seats (third row removed).
* **Versatility:** One middle seat (sliding door side) can be removed to create a 2+1 layout with space for a large compressor fridge.
* **Night mode:** After swiveling the seats 180 degrees, two bed sections are unfolded. They rest on dedicated supports with legs attached to the main unit and on the swiveled seats.
* **Sleeping area:** approx. 160 x 200 cm.
